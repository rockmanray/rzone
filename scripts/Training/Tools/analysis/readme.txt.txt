/*
			analysis of 8 direction rotation to estimate the prediction rate of CNN net
*/

1. estimated data should be collected to a directory, put under ./candidateDB/   e.g.  ./candidateDB/datas
2. to run the program, use ./analysis [dirName]  e.g. ./analysis datas
3. the result will be generated in the same directory with the program, named after the target directory  e.g. ./analysis datas  ->  datas.txt
